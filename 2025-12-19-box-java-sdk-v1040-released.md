# Box Java SDK `v10.4.0`のリリース

### バグ修正

* メタデータフィールド (読み取り) の定義にメタデータ階層を追加 (box/box-openapi[#572][1]) ([#1644][2]) ([`61235da`][3])
* `Retry-After`ヘッダーがない場合の`RetryAfter`関数を修正 (box/box-codegen[#903][4]) ([#1622][5]) ([`f135e2b`][6])

### 新機能と機能強化

* メタデータ階層APIを追加 (box/box-openapi[#569][7]) ([#1630][8]) ([`d1e8924`][9])
* Box Signのテキスト入力の検証 (box/box-openapi[#568][10]) ([#1624][11]) ([`8c5b5c1`][12])
* `nullable`フィールドを省略可として処理 (box/box-codegen[#906][13]) ([#1634][14]) ([`cacc729`][15])
