# Box CLI `v2.7.0`のリリース

**新機能と機能強化:**

* 以前非表示になっていたコマンド`collaborations:add`、`shared-links:update`、`shared-links:delete`、`users:search`が使用可能になりました ([#211][1])
* `filter_term`パラメータを`groups:list`に追加 ([#210][2])

**バグ修正:**

* プロキシ設定に関するバグを修正 ([#213][3])
