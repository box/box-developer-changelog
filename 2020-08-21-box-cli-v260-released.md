# Box CLI `v2.6.0`のリリース

[変更ログ][1]

* Zip機能の追加 ([#203][2]) [`1953639`][3]

* Windowsでレポートを保存する際に発生するファイル名の問題を修正 ([#195][6]) [`31853d3`][7]
