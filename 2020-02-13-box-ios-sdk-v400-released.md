# Box iOS SDK `v4.0.0`のリリース

Box iOS SDK

**重大な変更:**

* タスク割り当ての`status`フィールドを文字列から列挙型に変更
* `macOS`、`tvOS`、および`watchOS`のサポートを削除

**新機能と機能強化:**

* 認証セッションクラス`OAuth2Session`、`SingleTokenSession`、`DelegatedAuthSession`を公開
