# ChatGPT for remote Box MCP server released

<!--alex ignore -->

We've added the ability to [integrate ChatGPT with the remote Box MCP server][gpt-remote-box-mcp].
This allows ChatGPT to securely access Box content through a custom Model Context Protocol (MCP)
connector using OAuth and the MCP.
