# Users APIでマーカーベースのページ割りがサポート対象に

[Users API](e://get_users)で[マーカーベースのページ割り](g://api-calls/pagination/marker-based)がサポートされるようになりました。マーカーベースのページ割りとは、通常のオフセットベースのページ割りに代わるもので、多数のユーザーを有する会社のユーザーを取得する代替手段を提供します。
