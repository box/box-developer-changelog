# Box iOS SDK `v5.0.0`のリリース

**重大な変更:**

* ページを返してロジックを簡素化するようPagingIteratorを更新 ([#737][1])
* BoxClientの不適切な用語フィールド`collaborationWhiteList`を削除し、代わりに`collaborationAllowList`を使用 ([#790][2])

**新機能と機能強化:**

* 不適切なイベントタイプを置き換え ([#785][3])
* SignAPIのサポートを追加 ([#792][4])
