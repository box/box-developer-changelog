# Box Node SDK `v10.5.0` released

### Bug Fixes

* `retryAfter` has been converted from seconds to milliseconds for `setTimeout` ([#1356][1])

### New Features and Enhancements

* `canPublicSharedLinkBeCreated` has been added to `Hubs` ([#1367][2])
* Configurable timeouts have been added to SDKs ([#1360][3])
