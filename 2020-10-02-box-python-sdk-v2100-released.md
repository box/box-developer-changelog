# Box Python SDK `v2.10.0`のリリース

**新機能と機能強化:**

* メタデータテンプレートの`copyInstanceOnItemCopy`フィールドのサポートを追加 ([#546][1])
* `action`パラメータと`completion_rule`パラメータを使用したタスクの作成を許可 ([#544][2])
* 圧縮機能を追加 ([#539][3])

**バグ修正:**

* コラボレーションロールを所有者に更新する際のバグを修正 ([#536][4])
* `ints`を項目IDとして渡すことを許可 ([#530][5])
